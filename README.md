# Flow Launcher Theme Installation

Follow these steps to install this Flow Launcher theme:

## 1. Get the Theme File

Clone this repository **or** copy the theme `.xaml` file directly.

## 2. Open Flow Launcher Settings

1. Launch **Flow Launcher**
2. Go to **Settings → Appearance → Theme Folder**
3. Create a new `.xaml` folder.

## 3. Replace the New Theme File

1. After creating a new theme, Flow Launcher will open the **Themes folder**
2. Paste the downloaded `.xaml` file into this folder
3. Overwrite the default file if needed

## 4. Apply the Theme

Return to Flow Launcher and select the theme from the **Themes** list.
